# Holo-Constrictor
A language that is based on python but coded in English like language (to use the language an extension is given so pleasee when you are coding install the extension!)
Well to use it
* clone the repository and unzip the file
* star the repository
* open vs code and try to open the folder
* press Ctrl+Shift+X to open extensions and click the three dots (...)
* click "Install from VSIX" so you can use the extension
* locate the .vsix file and install the language extension and later install the debugger extension (provided in the folder)
